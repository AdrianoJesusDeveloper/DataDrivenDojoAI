# 🎯 Visão do Projeto

O **Data Driven Dojô AI** é um ecossistema de agentes inteligentes desenvolvido para auxiliar profissionais de tecnologia em sua evolução técnica e profissional.

O projeto une:

- 🤖 Engenharia de Inteligência Artificial
- 📊 Ciência de Dados
- ☁️ Computação em Nuvem
- 🎥 Produção de Conteúdo Técnico
- 💼 Desenvolvimento de Carreira
- 📚 Gestão do Conhecimento
- 🧠 Memória Inteligente
- 🏯 Filosofia do Data Driven Dojô

Nosso objetivo é construir um sistema modular, escalável e bem documentado, servindo como laboratório de aprendizado e portfólio profissional.

# 🚀 Como executar

Clone o projeto

```bash
git clone https://github.com/SEU-USUARIO/DataDrivenDojoAI.git
```

Entre na pasta

```bash
cd DataDrivenDojoAI
```

Crie o ambiente virtual

```bash
python -m venv .venv
```

Ative

Windows

```bash
.venv\Scripts\activate
```

Linux/macOS

```bash
source .venv/bin/activate
```

Instale as dependências

```bash
pip install -r requirements.txt
```

Execute

```bash
python src/main.py
```
