# ⚔️ Bem-vindo ao Data Driven Dojô

> **"Código resolve problemas. Arquitetura constrói soluções. Conhecimento compartilhado transforma carreiras."**

O **Data Driven Dojô AI** é um ecossistema Open Source criado para documentar, desenvolver e compartilhar a construção de um sistema de Inteligência Artificial do zero.

O projeto integra:

- 🤖 Engenharia de Inteligência Artificial
- 📊 Ciência de Dados
- ☁️ Computação em Nuvem (AWS)
- 🧠 Engenharia de Machine Learning
- 💼 Desenvolvimento de Carreira
- 🎥 Produção de Conteúdo Técnico
- 📚 Gestão do Conhecimento
- 🏯 Filosofia do Data Driven Dojô

Este repositório representa não apenas um software, mas toda uma jornada de aprendizado documentada desde o primeiro dia.

---

# 🎯 Missão

Construir um ecossistema de agentes inteligentes capazes de auxiliar profissionais de tecnologia em sua evolução técnica, profissional e na criação de conteúdo de alto nível.

Ser uma referência internacional em conteúdo sobre Inteligência Artificial, Ciência de Dados, Cloud Computing e Engenharia de Software, inspirando profissionais por meio da documentação transparente de uma jornada real de aprendizado.

# 🚀 Como executar

Clone o projeto

```bash
git clone https://github.com/SEU-USUARIO/DataDrivenDojoAI.git
```

Entre na pasta

```bash
cd DataDrivenDojoAI
```

Crie o ambiente virtual

```bash
python -m venv .venv
```

Ative

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

Instale as dependências

```bash
pip install -r requirements.txt
```

Execute

```bash
python src/main.py
```
