# 🏯 Arquitetura do DojoSenseiAI

## Objetivo

O DojoSenseiAI é um ecossistema de agentes especializados.

Cada agente possui uma responsabilidade única.

O Sensei atua como um roteador inteligente, identificando a intenção do usuário e encaminhando a solicitação para o agente mais adequado.

---

## Princípios

- Responsabilidade única
- Código modular
- Fácil manutenção
- Escalabilidade
- Reutilização
- Documentação contínua
- Arquitetura orientada a agentes

---

## Fluxo

Usuário

↓

Agente Especializado

↓

Resposta
