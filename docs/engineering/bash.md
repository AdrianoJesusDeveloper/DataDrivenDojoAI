| Comando      | O que faz                 | Quando usar                      |
| ------------ | ------------------------- | -------------------------------- |
| `pwd`      | Mostra o diretório atual | Quando não souber onde está    |
| `ls`       | Lista arquivos e pastas   | Antes de criar ou mover arquivos |
| `cd pasta` | Entra em uma pasta        | Navegação                      |
| `cd ..`    | Volta um nível           | Navegação                      |
| `mkdir`    | Cria diretório           | Estrutura do projeto             |
| `touch`    | Cria arquivo vazio        | Documentação e código         |
| `rm`       | Remove arquivos           | Limpeza                          |
