# Data Driven Dojo AI

## Arquitetura Geral

O projeto será composto por um agente principal chamado Sensei AI.

O Sensei é responsável por:

- entender o pedido do usuário;
- decidir qual especialista deve atuar;
- coordenar especialistas;
- revisar a resposta final;
- entregar uma resposta organizada.
